# Nest

An example [Nest](https://nestjs.com/) application that can be deployed to **Hostman Cloud Apps** with no configuration.

:tada: [Demo]()

:rocket: [Create your App](https://hostman.cloud/my/apps/create)

:books: [Hostman Cloud Apps Documentation](https://hostman.com/docs/app-platform/)

## <a name="dev"></a>Running the project locally

```bash
# install dependencies
npm install

# run in dev mode with hot reload
npm run start:dev

# build the application
npm run build

# run the application
npm run start:prod
```
